\# Task Manager App ✅



A simple fullstack Task Manager built with:

\- \*\*Frontend:\*\* React + Vite

\- \*\*Backend:\*\* FastAPI (Python)

\- \*\*Database:\*\* SQLite



\## Features

\- Add, edit, delete tasks

\- Mark tasks as Completed ✅ or Pending ⏳ (toggle switch)

\- Simple \& clean UI

\- REST API integration
